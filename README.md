# MedShakeEHR-Ansible

En cours de développement, ne pas utiliser en production.

- Installation d'apache
- Installation de MariaDB
- Installation de PHP
- Installation de MedShakeEHR
- Sécurisation d'apache
- Sécurisation de PHP
- Sécurisation de MariaDB
- Sécurisation de l'OS
